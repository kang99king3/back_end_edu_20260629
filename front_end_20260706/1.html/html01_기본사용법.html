HTML5 기본 사용법

웹 페이지의 뼈대를 구성하는 HTML의 핵심 개념과 필수 태그 가이드

HTML이란 무엇인가요?

HTML(HyperText Markup Language)은 웹 페이지를 만들기 위한 표준 마크업 언어입니다. 제목, 단락, 목록, 이미지 등 브라우저가 화면에 표시할 구조적 콘텐츠를 정의합니다.

기본 골격 코드 구조
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>문서 제목</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>첫 번째 문단입니다.</p>
</body>
</html>

자주 사용하는 핵심 태그

HTML 문서 구성 시 가장 기본적이고 필수적으로 사용되는 엘리먼트들입니다.

  • <h1> ~ <h6>

    제목(Heading)을 나타내며, 숫자가 작을수록 중요한 핵심 제목을 의미합니다.

  • <p>

    단락(Paragraph)을 정의하며, 본문 텍스트를 작성할 때 주로 사용합니다.

  • <a>

    하이퍼링크를 생성합니다. href 속성을 통해 이동할 경로를 설정합니다.

  • <img>

    이미지를 삽입합니다. src(경로)와 alt(대체 텍스트) 속성이 필수입니다.

  • <div> / <span>

    스타일링이나 스크립트 적용을 위해 영역을 묶어주는 무색무취의 컨테이너 태그입니다.

  • <ul> / <ol> / <li>

    순서가 없는 목록(ul)과 순서가 있는 목록(ol)을 각각 정의합니다.
